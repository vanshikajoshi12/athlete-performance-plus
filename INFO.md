# Sports Monitoring App
A prototy[pe app for monitoring athletes' heart rate, step count, accuracy, speed, stamina, diet, exercise plan, and to-dos.
## Features
Athlete data monitoring
Diet suggestions
Exercise planner
To-do list generation
## Structure
'data/athletes.csv'": Health and performance sample data
'src/monitor.py': Data reader
'src/diet.py': Diet planner
'src/exercise.py': Exercise suggestion
'src/todo.py': Athlete to-do lists
## How to use 
Place all the files as shown in the structure
Run scripts from 'src' folder using Python 3
